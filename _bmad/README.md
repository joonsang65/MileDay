# BMAD Lite Workspace

This directory is reserved for BMAD Lite source material and reusable planning inputs.

Current project artifacts are stored under `_bmad-output/`:

- `_bmad-output/planning-artifacts/product-brief.md`
- `_bmad-output/planning-artifacts/prd.md`
- `_bmad-output/planning-artifacts/architecture.md`
- `_bmad-output/planning-artifacts/epics.md`
- `_bmad-output/implementation-artifacts/stories/`
- `_bmad-output/implementation-artifacts/sprint-status.yaml`

Do not place generated run results here. Runtime artifacts for future local LLM evaluation runs should be written under `artifacts/runs/{run_id}/` as defined in the architecture document.
