## 1. 마일스톤 계획

요구사항 정의 내용을 기반으로 MileDay의 개발 마일스톤을 정의한다.

### 1. 개발 환경 구축하기

1. FastAPI 기반 백엔드 구성
2. Supabase DB를 사용해서 테이블 스키마 구축
3. RLS 적용 및 검증 로직 구축
4. 예외 처리 및 에러 검증 로직 구축

### 2. 회원가입 기능 구현

1. 회원가입, 로그인, 로그아웃 기능 구현
2. 사용자 정보 기반 RLS 동작 검증
3. JWT 기반 인증 토큰 검증 로직 구현

### 3. 목표 관리, 일정 반복 로직 구현

1. 목표 CRUD 구현
2. 반복 기능 구현
3. 색상 필드 구현

### 4. 마일스톤 관리 로직 구현

1. 마일스톤 CRUD 구현
2. 목표 색상과 겹치지 않고 어떻게 색상 관리할지 고민

### 5. 캘린더 UI 구현

1. 캘린더 핵심 기능 구현
2. 특정 날짜 조회 및 목표 내용 확인 로직 구현
3. Today List 구현 및 달성 여부 확인 기능 구현

### 6. 위젯 창으로 표시

1. 구현 내용을 기반으로 데스크톱 위젯으로 구현
2. 창 크기, 위치 등 레이아웃 변경 기능 구현

### 7. 릴리즈 직전 단계

1. 컴퓨터를 켜면 자동으로 실행되도록 설정
2. 엣지 케이스에 대한 에러 처리 및 보완
3. UI 최종 개선
4. 파일럿 테스트

### 8. 기능 고도화

1. 투명도 조절 기능
2. 챗봇 기반 일정 파싱 및 자동 등록 기능
3. 외부 캘린더 연동 및 모바일 앱 구현

### 9. 배포 및 기술 문서 작성

1. 앱 패키징 및 배포
2. 최종 결과 보고서 작성
